Movie Ticket Booking System
This is a simple python movie ticket booking system.
It allows users to book tickets by entering a movie title, customer name, and number of tickets.
The Booking is saved into a text file (booking.txt) and includes input validation using custom exceptions.

features:
-Book movie tickets
-Input Validation(no negative or zero tickets)
-Maximum ticket limit(1-20 tickets)
-Custom exception handling
-Save bookings to file("booking.txt")
-view saved bookings
-----------

Technologies used:
-python3
-file handling
-object-oriented Programming(oop)
-exception Handling

-----------
how to run:
-open the project folder
-run the programm:
    -python main.python
    -check saved data in (bookings.txt)
